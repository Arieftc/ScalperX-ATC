# ScalperX ATC

ScalperX ATC adalah sistem backend trading otomatis berbasis AI terintegrasi untuk platform MT5, MT4, dan Indodax. Dirancang untuk mencapai akurasi sinyal hingga 95% melalui konfirmasi multi-indikator berbasis cloud dengan fitur:

- Auto Trading 24/7
- Dukungan multi-platform (MT5, MT4, Indodax)
- AI Commander, AI Spy, Ghost Sync Protocol, Early Warning C4SR
- Sistem hedging adaptif & multi-entry AI
- Backend sinergi real-time cloud-native
- Tidak tergantung pada versi Android

**Status saat ini:**  
Live untuk akun demo MT5 HFMarkets (48783800) dan integrasi backend Indodax.

---

**Catatan:**  
Semua source code dan modul backend akan ditambahkan secara bertahap dalam struktur terorganisir. Setiap komponen memiliki dependensi sistematis dan saling terhubung secara modular.
